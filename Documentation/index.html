Getting started

Welcome

Hello there! Thanks for buying Quick Lend. Quick Lend is a Landing page template with solid design especially crafted to help you built a Loan agency Landing Page website with huge variety of options.

This Template is suitable for all types of loan Agency Landing Page. This template built using Twiiter's Bootstrap v4.5.3 Framework, Google Fonts, jquery -v1.12.4, jQuery UI - v1.12.1. It is highly recomended that you are familiar in a way with at least HTML5 / CSS3 / Basic JQuery/SASS.

So let's jump directly to it, and start everything from scratch.

Installation

Install the template via ftp :

  1. Connect to your website host, using some of many FTP clients, for example FileZilla.
  2. Go to your default server folder, and copy Quick Lend into that folder.

Setting your fonts

To use google fonts, head over to Google's Fonts and add whichever font you want to your collection. Once you finished, follow the instructions and place the generated link in index.html file.
The link should look like this : 

<link href="https://fonts.googleapis.com/css2?family=Open+Sans:wght@300;400;700&display=swap" rel="stylesheet"/>
Next, just replace the font family, with the font family you just copied. Like: 
 body {  font-family: "Open Sans", sans-serif;}

Using Quick Lend.

Changing favicon

Favicons are located in images folder. You can simply change them by replacing it with your own favicon , and include this code to the HTML Documents: 

	
<link rel="apple-touch-icon" sizes="57x57" href="images/favicon/apple-icon-57x57.png"/>

Changing logo

To change logo, simply replace the logo.png at images/logo.png. Please make sure that your logo size is 160x60 px and in png format

<img src="images/logo/logo.png" alt="Company Name" class="logo" />
